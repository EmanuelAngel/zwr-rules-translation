SUBMISSIONS

  • To prevent mistakes and rejections, players should always read the rules carefully.
  • Make sure your submission includes all requirements which are listed under each specific challenge's rules.
  • As of May 12th 2024, for all flawless submissions, you must provide an indication of where your flawless run ends in the submission message box as a timestamp.
  • Always double check if you have chosen the right game, map, category and challenge, and if all players are listed.
  • Every submission must contain full gameplay otherwise it will be denied.
  • Every submission must contain sufficient game audio otherwise it will be denied.
  • After a game is approved, the footage is expected to stay accessible for the record to remain on the leaderboards. If the footage ends up missing or private for any reason, that record is no longer considered valid and will be removed from the leaderboards.

GENERAL RULES

MAKE SURE TO READ THE CHALLENGE SPECIFIC RULES ON EACH RESPECTIVE LEADERBOARD PAGE!

  • Players caught cheating will be subject to the CONVICTED CHEATER title. See below for more information.
  • FOV on PC in older games without an FOV slider (for newer games you can set FOV with a built-in option) is ALLOWED.
  • Changing FOV on games without an FOV slider (for newer games you can set FOV with a built-in option) is NOT ALLOWED on CONSOLE ONLY. Console must play Default FOV.
  • Only 1 player is required to be actively playing during coop games; however, if a player is AFK (not playing) for the entire duration of a game, they will be removed from the record and replaced with an AFK account.
  • Duplicating wonder weapons is NOT ALLOWED unless stated otherwise.
  • Glitches, bugs, or exploits with the exception of specific ones which were voted as allowed by the community, are NOT ALLOWED.
  • Round skipping of any kind is NOT ALLOWED unless stated otherwise.
  • Abusing insta-kill melee weapons (for example by locking a melee key and going AFK for hours) is NOT ALLOWED.
  • Farming records by playing with other players and getting a tie with your own record is NOT ALLOWED. One tie with a different player is acceptable, but anything more will result in the removal of games.
  • Co-op Splitscreen is ALLOWED.
  • Creating alternate profiles dedicated to only world records to ensure a perfect ratio is NOT ALLOWED. Players are encouraged to submit all records to one profile

CHEATING

CONVICTED CHEATER

  • Any player caught cheating will receive a line through their name and the CONVICTED CHEATER title.
  • Being a CONVICTED CHEATER simply means that you will no longer be able to submit solo records, or any co-op records again.
  • All cheaters will be permanently banned from submitting new records.
  • All former Convicted Cheaters are no longer allowed to submit solo games.

Immediate Convictions

  • If a player is caught injecting a game to gain an unfair advantage with the use of a modification, they will immediately be given the CONVICTED CHEATER title.
  • If a player is caught splicing or altering the speed of a speedrun, they will immediately be given the CONVICTED CHEATER title.
  • If a player is caught using blatant cheats such as God Mode or any other obvious modifications with clear malicious intent, they will immediately be given the CONVICTED CHEATER title.

  • If submitting a game with multiple videos, the end of the game needs to be submitted as the main link. All other links should be submitted in the "Additional links" section.

Platforms

  • YouTube - All footage must be PUBLIC or UNLISTED.

Easter Egg General Rules:

Across all titles, for Easter Egg World Record Speedruns, the footage of ALL players in the game is REQUIRED.  

The following rules for Easter Egg Speedruns were finalized on October 5th, 2023. The main results can be viewed here

The following rules for Easter Egg Speedruns APPLY ONLY for the following titles: Black Ops 1, Black Ops 2, Black Ops 3, Black Ops 4, Cold War, and Vanguard.

    Submission Rules:

  • Please check the main submission rules.
  • Additional requirements if you run is within Top 3:
    • Any top 3 CO-OP run will be required to upload recordings for all players in the game. This is the rule as of 1/9/2024, meaning any runs before that will remain on the leaderboards, regardless if they meet the rule or not.
    • Any top 3 run achieved off stream, will be required to upload session gameplay (proof of resets). This is meant to help prove that the run is legitimate. This would mean that a player provides at least two videos:
      • The submitted run (which can be edited with text, effects, etc, as long as it does not make the actual run unwatchable).
      • The unedited, raw gameplay. This should be the entire recorded session. This must have the gameplay and the gameplay audio (the video can be unlisted). Proof of restarts NEED to be provided as an additional link in the submission process OR in the description of the video as a link.
      • In Co-op games, at least the host must upload session gameplay.
    • The proof of resets video should not be deleted even after the run is accepted.
    • Keep in mind that staff reserve the right to ask for more footage if needed.
    • As of 04/26/2024, theater mode submissions will no longer be allowed to be used as the main footage of a submission. Players are encouraged to stream or record the raw footage of their game. Although it is recommended for all players to have raw footage for coop submissions, theater mode is acceptable unless you are the host. The host will still be responsible for pregame lobby/showing elixirs and proof of restarts.

    Custom Textures:

  • If you want to be safe, just don’t change anything. 
  • The following texture changes are ALLOWED (Gun textures/camos, viewarms/gloves, and HUD elements (Perk icons. etc))
  • The following texture changes are NOT ALLOWED (Zombie models, map textures, player models, buildable parts, and effects such as explosions, power ups, etc)
  • Be mindful that some textures will change more than one thing. It is your responsibility to make sure that a texture change doesn’t also result in using a banned texture change.
  • Custom Audio files are NOT ALLOWED.

    Timing Method:

  • EE Speedruns use In-Game Time (IGT) as the timing method. This means that if you pause the game, the timer will also pause.

BREAKS & DISCONNECTS

  • The player must reconnect in the same round they disconnect before the next round starts, or the game will be void after the round changes (also applies if a player leaves the game). e.g. player disconnects on 20 and game goes to 21, posted round will be 21.
  • Players must continue to use the same loadout before disconnecting, if joining back, they are NOT ALLOWED to switch their GobbleGums from their previous loadout (BO3).
  • Any form of sleeper is ALLOWED for a coop game. Examples are the use of sit-in glitches, having a player disconnect and reconnect, holding zombies in windows, zombies under the influence of an EMP, and staying out of the map as long players stop killing zombies or doing EE steps. However, using cheats to perform a sleeper (such as God Mode or No Clip) is NOT ALLOWED.

STAFF

  • Staff Members are the people who do volunteer work for the zombies community. Their responsibility is verifying submissions, they determine if games are following the rules and meeting requirements.
  • If a Staff Member denies a submission, there is always a reason for it. If the player did not receive an email with the reason he can always reach out to the administration.
  • Staff Members can occasionally make mistakes. If you see a mistake, you can report it using the "Report a problem" button, you can find it at the bottom of a page.
  • The verification process may take some time. The status of your submission can be checked HERE
  • You can contact staff and administration through the ZWR Discord or email support.
